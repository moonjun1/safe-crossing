# 서비스 개발 기획서

## 1. 서비스명
안심 횡단 - 느린 보행자를 위한 교차로 안전 분석

## 2. 서비스 개요
고령자, 교통약자 등 느린 보행자의 보행속도에 맞춰 전국 교차로의 횡단 안전도를 분석하는 서비스입니다. 교차로의 차로폭과 보행신호 시간을 AI가 분석하여, 보행속도별로 안전하게 건널 수 있는지 판단합니다.

## 3. 활용 데이터
### 필수 활용 데이터 (전국 통합개방데이터)
- **교통안전 신호등 실시간 정보 (2종)**
  - 교차로 맵 정보 (crsrd_map_info): 교차로ID, 교차로명, 위경도, 차로폭, 제한속도
  - 신호제어기 신호잔여시간 정보 (tl_drct_info): 8방향별 보행신호 잔여시간/상태

### 추가 활용 기술
- Leaflet: 교차로 위치 지도 시각화
- Recharts: 위험도 통계 차트
- 클라이언트 사이드 위험도 계산 엔진

## 4. 서비스 주요 기능

### 4-1. 보행속도별 교차로 위험도 분석
- 핵심 공식: 위험도 = (차로폭 / 보행속도) / 보행신호시간
- 위험도 > 1.0: 위험 (횡단 시간 부족)
- 위험도 0.7~1.0: 주의
- 위험도 < 0.7: 안전
- 보행속도 프리셋: 일반인(1.2m/s), 고령자(0.8m/s), 휠체어(0.6m/s)

### 4-2. 실시간 속도 시뮬레이션
- 슬라이더로 보행속도 조절 시 교차로 위험도 즉시 재계산
- API 재호출 없이 클라이언트에서 실시간 계산
- 속도 변경 시 인사이트 문구: "고령자 속도로 변경하면 위험 교차로가 3개→8개로 증가합니다"

### 4-3. 교차로 위치 지도
- Leaflet 기반 지도에 교차로 위험도별 색상 마커
- 빨강(위험), 노랑(주의), 초록(안전), 회색(분석 불가)
- 마커 클릭 시 상세 팝업

### 4-4. 위험도 통계 차트
- 도넛차트: 현재 속도 기준 위험/주의/안전/분석불가 비율
- 바차트: 보행속도별(0.6, 0.8, 1.0, 1.2m/s) 위험도 비교

### 4-5. 전국 위험 교차로 TOP 10
- 주요 7개 지역 교차로 데이터 병렬 조회
- 고령자 기준 가장 위험한 교차로 순위 제공
- "대한민국에서 고령자가 가장 건너기 어려운 교차로" 인사이트

### 4-6. 검색/필터 및 공유
- 교차로명 검색, 위험도별 필터링
- URL 공유 (지역+속도 파라미터 포함)

## 5. 기대효과
- **안전**: 고령자/교통약자의 횡단 사고 예방을 위한 데이터 기반 분석
- **정책 지원**: 위험 교차로 개선 우선순위 도출에 활용 가능
- **사회적 가치**: 교통약자 이동권 보장을 위한 기초 데이터 제공
- **장애인의 날(4/20) 시의성**: 교통약자 안전에 대한 사회적 관심 제고
